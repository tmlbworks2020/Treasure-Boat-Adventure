-----------------------------------------------
   Treasure Boat Adventure - ver1.4
-----------------------------------------------

帆掛け船を操縦するゲームです。
宝貝をゲットするなどポイントをアップして
ゲームクリアを目指します。


[起動方法]
ブラウザ(Chrome)で treasure_boat_adv_start.htm を開きます。


[動作環境(確認したもの)]
64 bit版  Windows10 
32 bit版  Windows7

[起動に必要なソフト]
Google Chrome

---------------------------
         操作方法
---------------------------

1) 画面下部の操作ボタンを押して操作します。
  (Nは停止、Rは後進）

2) 風向きを示す矢印があり、ボートの速度と方向に
   影響します。
     
3) @ボタンはポイントがある時に、エサをまいて
   海獣を撃退します。
   
4) ポイントは円形範囲内にいる間と宝貝を取得した際に
   発生します。
